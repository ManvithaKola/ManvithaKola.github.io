Image Captcha Breaker

ManvithaKola/Captcha-Image-Breaker

Convolutional Neural Network Based Image Captcha Breaker using tensorflow-lite.

The Image captchas are generated using python package captcha which rotates,scales and translates the symbols and also adds noise to the image making it difficult to read.

The captcha is fed to a Convolutional Neural Network after various pre-processing steps,which then predicts the captcha with over 95% accuracy. Since the prediction is to be done on an edge device(raspberryPi), tensorflow model is converted to a tensorflow-lite model as it is light-weight and has low-latency. Tensorflow Lite Converter converts a pre-trained Tensorflow model to Tensorflow Lite flat buffer file(.tflite).

For further details please read the README.md file on my GitHub.

Vehicular Network Simulation on RaspberryPi

ManvithaKola/VANET-Simulation-on-Rpi

5 different vehicular network scenarios are generated in a VANET with 25 devices using pubnub to establish connections between devices(RaspberryPi's) . 10 vehicles, 5 Traffic signals, 5 RSU's and 5 Parking slots are simulated on RPi's. The 5 different scenarios include

1. Traffic Signal : Vehicle receives the message from the near-by traffic signal while it is in the vicinity of the signal and takes an appropriate action .

2. Parking Slot : Vehicle sends a message to the parking slot to enquire the current status and based on the reply, it takes an appropriate action . If the parking slot is empty, vehicle moves into the parking slot and the message is relayed back to the parking slot which changes slot to 'Occupied'

3. Reporting Accident: When a vehicle comes across an accident, it reports it to nearby RSU

4. RSU-Vehicle : When there is an accident reported to RSU, it passes on that information to all the remaining RSU's and when a vehicle is about to enter the accident lane, RSU sends a message informing the accident and the vehicle changes the lanes accordingly

5. RSU-Ambulance : When an accident is reported to an RSU, it alerts the nearby ambulance to move to the accident spot

6. RSU-drone : When a drone passes by an accident scene and spots it using it's sensors, it passes the information to the RSU on its trajectory, which inturn alerts the remaining RSU's

To understand the scenarios better, watch the animations here: VANET-Scenarios folder.

For further details please read the README.md file on my GitHub.
